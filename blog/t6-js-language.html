Sprint 6 Technical blog

In the JavaScript language, describe all the use cases for each of:

Parentheses ()

Used to pass values and parameters as 'arguments' through functions. Also used to structure values or equations for statements, such as if statements. Also used to call functions independent of arguments or values, such as boardNewGame().

Brackets []

Used to create arrays with all values in the array becoming elements. Also used to call, index, or access values within an array. An example can be to access a value, and then pass it to another array, such as:

for (let i = 0; i < board.cells.length; i++)
board.cells[i].surroundingMines = countSurroundingMines(board.cells[i]);

This is a loop, ending when the value of i reaches board.cells.length.

Each time i is passed with an increment value, the value of i is passed through board.cells.surroundingMines run the countSurroundingMines function with the same value passed through i.

Braces {}

Creates objects, with values in them created as properties. The indexes of those properties can be defined within the braces as well. For example:

var myOfficeKitchenCutleryDraw = {
fork: 4
knife: 8
bigSpoon: 6
teaSpoon : 0
}

Single quotes ''

Used to create a string, with the contents of that string being changeable by functions>. Can be used to reference a string elsewhere on a partial basis.

Double quotes ""

Used to create a string, with the contents of that string being immutable (i.e. cannot be modified). Can be used to reference the same string, but it must be a one-for-one match.
